PorrAytos

Aplicación web desarrollada para la gestión de una porra particular.

Con dicha porra podemos gestionar las apuestas, consultar las apuestas realizadas, consultar el bote acumulado.

Características:

  • Aplicación Web desarrollada en HTML5, PHP y CSS3.
  • Usuario administrador con opciones adicionales para recargas de creditos y modificacion de datos de los participantes.
